Hello from Insta Backgrounds!

iOS shortcut to add solid backgrounds to your stories

Ever tried posting a photo that isn’t 16:9 on your Insta story? That blurred background Instagram adds is... so boooring.

Solid backgrounds look way cleaner, but making them manually is a pain: add one image, paint it white, add another, repeat… Ugh. So I made a shortcut to handle it for you. Simple, fast, done.

How to use:

  1. Add the Shortcut on your iPhone: Get Insta BG
  2. Open Shortcuts → Insta BG → ℹ️ → Show in Share Sheet → turn ON
  3. Open Photos, select the images you want, then Share → Insta BG
  4. Pick a background you like
  5. …wait a moment…
  6. Done! Your photos with backgrounds appear in the album "Insta Backgrounded"

Advanced options:

  1. Edit the set of available backgrounds: Open the folder Insta Background Templates on your iPhone and add or remove images. Each image should be 1080x1920. If there’s only one, it will be applied automatically; otherwise, you’ll choose each time.
  2. Adjust the width of the framed image: By default, the image width is 960px (9/10 of 1080). To change it: Shortcuts → Insta BG → ... → change the value 960 to what you want.
