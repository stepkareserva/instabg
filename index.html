Insta Backgrounds

A clean iOS shortcut to fix blurry Instagram Stories

Why this matters

When you post photos with non-standard aspect ratios to Instagram Stories, you get those ugly blurred edges. Sure, you could use Instagram's manual hack, but that’s clunky and slow.

A simple solution

I made a shortcut that does all the work for you. Add solid backgrounds to your photos in just a few taps.

Install the Shortcut

How it works

Setup

Tap Install the Shortcut above on your iPhone. It will open in Shortcuts — just tap Skip Setup.

Run it once and allow permissions (download backgrounds, photo access, sharing).

Use from Photos

Open Photos, select an image, tap Share, and choose Insta BG.

Select and Save

Pick a background color, wait a moment, and find your new image in the Insta Backgrounded album — ready for Instagram.

Advanced Options

Customize Shortcut

Open Shortcuts → Insta BG → … → i → Customize Shortcut. Here you can change the frame width (default is 1080).

Custom Backgrounds

Add your own 1080x1920 images to the Insta Background Templates album. If you have no plans for using some backgrounds, remove it from this album.
